수학 선택과목 나침반

실행 방법
1. index.html을 더블클릭하면 인터넷 연결 없이 실행됩니다.
2. GitHub Pages에 배포할 때는 기존 index.html을 이 파일로 교체한 뒤 커밋·푸시하면 됩니다.

이번 버전 주요 기능
- 1학년·2학년별 선택과목 추천 알고리즘
- 넓은 진로 분야 → 구체적인 희망 학과 → 희망 직업·활동 질문
- 희망 학과에 따른 수학 과목 적합도 가중치 반영
- 실제 대학 교과과정·권장이수과목을 참고한 학과별 과목 활용 설명
- 인공지능 수학·기하·미적분Ⅱ 학습 내용 팝업
- 업로드된 선택과목 안내 PDF를 바탕으로 핵심 개념, 학습 활동, 탐구 예시 제공
- 결과 복사, 인쇄/PDF 저장, 교사용 설정, 추천 축포 애니메이션

주의
이 결과는 합격 가능성이나 대학의 공식 가산점을 판정하지 않습니다.
희망 대학의 최신 시행계획·모집요강과 학교의 실제 편제표를 확인한 뒤 상담 자료로 활용하세요.
